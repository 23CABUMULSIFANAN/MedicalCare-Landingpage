## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/MedicalCare-Landingpage.git
```

Move into the project folder:

```bash
cd MedicalCare-Landingpage
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm start
```

Open:

```bash
http://localhost:3000
```

## Build for Production

```bash
npm run build
```
